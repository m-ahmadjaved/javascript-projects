# 🎯 Guess My Number

A browser-based number guessing game built with vanilla JavaScript.

The player tries to guess a secret number between 1 and 20. Each wrong guess reduces the score, and the game tracks a highscore across rounds.

---

## 🚀 How to Play

1. Enter a number between 1 and 20
2. Click **Check** — you'll get a hint if you're too high or too low
3. Keep guessing until you find the secret number
4. Hit **Again** to reset and try to beat your highscore

---

## 💡 Concepts Covered

- DOM selection and manipulation
- Handling click events with `addEventListener`
- Implementing game logic (win/lose conditions)
- Dynamically manipulating CSS styles via JavaScript
- Tracking and updating highscore across rounds
- Refactoring with the **DRY principle** using reusable functions

---
