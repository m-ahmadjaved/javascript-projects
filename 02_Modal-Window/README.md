# Modal Window

A clean, interactive modal window built with JavaScript. Click a button to open a popup overlay, and close it by clicking the X button, clicking outside the modal, or pressing the Escape key.


## 🚀 Features

- Open modal via multiple trigger buttons
- Close modal by clicking the ✕ button
- Close modal by clicking the overlay/backdrop
- Close modal by pressing the `Escape` key
- Smooth show/hide using CSS class toggling

---

## 🛠️ Built With

- HTML5
- CSS3
- Vanilla JavaScript (ES6)

---

## 📚 Concepts Practiced

- DOM selection (`querySelector`, `querySelectorAll`)
- Event listeners (`click`, `keydown`)
- CSS class manipulation (`classList.add`, `classList.remove`, `classList.contains`)
- Looping through NodeList
- Keyboard event handling (`e.key`)
- Writing reusable functions (DRY principle)
