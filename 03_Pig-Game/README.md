# 🎲 Pig Game

A simple two-player dice game built with JavaScript, HTML, and CSS.

## 📖 About the Project

Pig Game is a fun dice game where two players compete to reach 100 points first.

### Rules

- Players take turns rolling a dice.
- If the dice roll is **2-6**, the value is added to the current score.
- If the dice roll is **1**, the current score is lost and the turn switches to the other player.
- A player can click **Hold** to add the current score to their total score.
- The first player to reach **100 points** wins the game.

## 🎮 How to Play

1. Click **Roll Dice**.
2. Keep rolling to increase your current score.
3. Click **Hold** to save your current score.
4. Avoid rolling a **1**, or you'll lose your current score and your turn.
5. First player to reach **100 points** wins.
